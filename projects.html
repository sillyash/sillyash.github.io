Image Manipulation

·

C++

·

link to project

·

Updated on 17/8/2024

A CLI for PPM image manipulation, made in C++. PPM images are notably used in the widely used GIMP software.

This was my second semester project, for algorithmics, matrixes and C++. The subject used PPM files because they're quite easy to read/write.

The CLI I made with my colleague uses knowledge in colors, pixels, blurriness, to make different image effects and manipulations available, like black and white, contrast, luminosity...

PyKemon

·

Python

·

link to project

·

Updated on 17/8/2024

This is a much older project : it dates back to my second year of highschool, where I was learning Python. Our teacher gave us a project where we could do whatever we wanted. So I decided to pair up with a colleague and make a Pokemon game.

We pretty much didn't know how to do a GUI so we decided to do a CLI Pokemon game. We also didn't have the time and resources to implement all the different abilities of pokemons so we decided to just simplify the game as a whole.

The game can be quite fun to play, even though it's just a single battle.
